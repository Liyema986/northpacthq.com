# Project Overview

## Vision
A comprehensive platform for tracking and improving tertiary student academic performance through data-driven interventions.

## Scope
**In Scope**: Student tracking, tutoring management, intervention coordination, reporting, multi-role access

**Out of Scope**: Mobile native apps, payment processing, video conferencing

## Key Stakeholders
- Students
- Parents
- Tutors
- Coordinators
- Administrators
- Management
- Funders
