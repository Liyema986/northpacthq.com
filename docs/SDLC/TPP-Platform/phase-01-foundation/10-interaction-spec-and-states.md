# Interaction Spec & States

## Screen States

### Loading State
- Skeleton placeholders
- Shimmer animation
- Progress indicators for long operations

### Empty State
- Illustration
- Helpful message
- Call-to-action button

### Error State
- Error icon
- User-friendly message
- Retry action

### Success State
- Success toast
- Data refresh
- Confetti animation (optional)
