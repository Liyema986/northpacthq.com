# GUI Design Specifications

## Color Palette
- Primary: red-600
- Success: green-500
- Warning: amber-500
- Error: red-500
- Neutral: slate-900/500/200

## Typography
- Heading: 2xl, bold
- Subheading: xl, semibold
- Body: base, normal
- Small: sm, normal

## Spacing
- Section: 24px (6)
- Component: 16px (4)
- Element: 8px (2)
